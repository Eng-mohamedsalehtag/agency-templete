# Agency Template

A professional agency website template built with HTML5, CSS3, and Bootstrap 5. This project showcases a modern, responsive design suitable for agencies, studios, and service-based businesses.

## Features

- **Responsive Design** - Fully responsive layout that works on all devices and screen sizes
- **Modern UI** - Clean and professional design with a sophisticated color scheme
- **Navigation Bar** - Fixed navigation bar with smooth scrolling
- **Hero Section** - Eye-catching header with background imagery
- **Bootstrap Framework** - Built on Bootstrap 5.3.8 for reliability and consistency
- **Font Awesome Icons** - Comprehensive icon library for visual enhancements
- **Google Fonts** - Custom typography with Montserrat and Roboto Slab fonts
- **Mobile-Friendly** - Fully optimized for mobile and tablet devices

## Technologies Used

- **HTML5** - Semantic markup structure
- **CSS3** - Custom styling with CSS variables for easy customization
- **Bootstrap 5.3.8** - Responsive grid system and components
- **Font Awesome 6.3.0** - Icon library
- **Google Fonts** - Professional font family selection
  - Montserrat (main font)
  - Roboto Slab (secondary font)

## Color Scheme

- **Primary Color**: #212529 (Dark Gray)
- **Secondary Color**: #ccbb4f (Gold)

## Installation

### Prerequisites

- Node.js and npm (for managing dependencies)
- A modern web browser

### Steps

1. **Clone the repository**

   ```bash
   git clone https://github.com/Eng-mohamedsalehtag/agency-templete.git
   cd agency-templete
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Open in browser**
   - Simply open `index.html` in your web browser, or
   - Use a local development server (e.g., Live Server extension for VS Code)

## Usage

### Customization

1. **Update Colors** - Modify CSS variables in `style.css`:

   ```css
   :root {
     --main-color: #212529;
     --secondary-color: #ccbb4f;
   }
   ```

2. **Add Content** - Edit `index.html` to add your agency information, services, portfolio items, and team members

3. **Replace Images** - Update image files in the `assets/img/` directory with your own images

4. **Modify Fonts** - Update Google Fonts links in the `<head>` section of `index.html`
